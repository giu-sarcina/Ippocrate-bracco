## Federated UNet

This is an implementation of the UNet model for segmentation of single or multiclass problems in the NVFlare environment. 

Compatible with NVFlare 2.5.2. 

### Description
2D/3D UNet model architecture and training for use in a federated setting with NVFlare.

To use in the Ippocrate NVFlare environment, install the requirements with 
```shell
pip install -r ./federated_UNET/requirements.txt
```
then run the experiment.
